# UnityWebGLKits
The Unity WebGL Platform Extension Kis.

## 在线示例
[https://psygames.github.io/WebGLKits/](https://psygames.github.io/WebGLKits/index.html)

## 功能列表

- 弹窗 (alert)
- 日志 (console.log)
- 设置窗口分辨率
- 执行JS代码 (eval)，能实现80%的JS交互功能，以上三种功能均可使用此功能实现。
- 打开本地文件 (input:file)
